Liquid Unit Conversion

Liquid Unit Conversion Help

Enter the liquid volume amount, choose the input unit, then choose the unit you want to convert to.

Supported units include gallons, milliliters, liters, ounces, cups, pints, and quarts.

The calculator converts the input amount to a common base unit, then converts that value into the selected output unit.
